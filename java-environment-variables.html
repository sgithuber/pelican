Java Environment Variables

Mon 05 October 2020

java enviroment variables settings for Windows systems

Keyword Value
CLASSPATH .;%JAVA_HOME%\lib\;%JAVA_HOME%\jre\lib
JAVA_HOME C:\Program Files\Java\jre1.8.0_241
JDK_HOME %JAVA_HOME%
JRE_HOME %JAVA_HOME%\jre

java enviroment variables settings for Linux systems

[root@localhost ~]# tail -n 3 /etc/profile
JAVA_HOME=/opt/java/jre1.8.0_241/
export JAVA_HOME
PATH=$PATH:$JAVA_HOME/bin

check current activated java's varsion

[root@localhost ~]# java -version
java version "1.8.0_241"
Java(TM) SE Runtime Environment (build 1.8.0_241-b07)
Java HotSpot(TM) 64-Bit Server VM (build 25.241-b07, mixed mode)

if multiple java versions has installed, using below method to select one of them

[root@localhost ~]# update-alternatives --install /usr/bin/javaws javaws /opt/java/jre1.8.0_241/bin/javaws 2
[root@localhost ~]# update-alternatives --config java
