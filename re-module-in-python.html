>>> import re
>>> result = 'Joe Biden Wins'
>>> search = re.search('wins',result)
>>> print(search)
None
>>> search = re.search('Wins',result)
>>> print(search)
<re.Match object; span=(10, 14), match='Wins'>

Conclusion

  • we can use re.search(args) to find a string which contains args.
  • it's case sensitive

but match() & fullmatch(), i really don't know what the purpose they exist for?
