how to synchronize time with ntp in linux

Wed 18 December 2019

the most famous ntp synchronize service all over the world i think maybe ntpool.

set timezone to Asia/Tokyo with timedatectl.

[root@localhost ~]# timedatectl set-timezone Asia/Tokyo
[root@localhost ~]# timedatectl
Local time: Wed 2019-12-18 00:51:19 JST
Universal time: Tue 2019-12-17 15:51:19 UTC
RTC time: Tue 2019-12-17 15:51:18
Time zone: Asia/Tokyo (JST, +0900)
System clock synchronized: yes
NTP service: inactive
RTC in local TZ: no

active timedatectl's ntp service.

[root@localhost ~]# timedatectl set-ntp 1
[root@localhost ~]# timedatectl
Local time: Wed 2019-12-18 00:51:25 JST
Universal time: Tue 2019-12-17 15:51:25 UTC
RTC time: Tue 2019-12-17 15:51:24
Time zone: Asia/Tokyo (JST, +0900)
System clock synchronized: yes
NTP service: active
RTC in local TZ: no

install and configure ntp service. 

[root@localhost ~]# dnf install ntp -y
[root@localhost ~]# systemctl enable ntpd
[root@localhost ~]# vim /etc/ntp.conf
server 0.jp.pool.ntp.org
server 1.jp.pool.ntp.org
server 2.jp.pool.ntp.org
server 3.jp.pool.ntp.org
[root@localhost ~]# systemctl restart ntpd
[root@localhost ~]# ntpq -p
