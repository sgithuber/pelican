Jekyll, a static site generator written with Ruby

Mon 05 October 2020

first of all, let's install the essential packages

[root@localhost ~]# dnf install make gcc gcc-c++ -y &

search for the package that provides command gem

[root@localhost ~]# dnf provides gem
rubygems-3.1.2-128.fc32.noarch : The Ruby standard for packaging ruby libraries
Repo        : fedora
Matched from:
Provide    : gem = 3.1.2-128.fc32

rubygems-3.1.2-130.fc32.noarch : The Ruby standard for packaging ruby libraries
Repo        : updates
Matched from:
Provide    : gem = 3.1.2-130.fc32

install ruby, ruby-devel and rubygems

[root@localhost ~]# dnf install ruby ruby-devel rubygems -y &

then install bundler and jekyll

[root@localhost ~]# gem install bundler jekyll

let's create a new site

[root@localhost ~]# jekyll new jekyll
[root@localhost ~]# cd jekyll/
[root@localhost jekyll]# bundle exec jekyll serve
