convert a string to a list

>>> sakura = 'Japanese cherry, which is commonly called sakura (桜 or 櫻; さくら).'

>>> temp = sakura.split(' ')
>>> list = []
>>> for item in temp:
>>>    list.append(item.strip(',;.()'))

>>> list
['Japanese',
 'cherry',
 'which',
 'is',
 'commonly',
 'called',
 'sakura',
 '桜',
 'or',
 '櫻',
 'さくら']

convert a list to a string

>>> string = ''
>>> for item in list:
>>>     string = string + item + ' '

>>> string.strip()
'Japanese cherry which is commonly called sakura 桜 or 櫻 さくら'
