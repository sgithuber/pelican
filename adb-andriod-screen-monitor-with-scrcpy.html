adb andriod screen monitor with scrcpy

Sun 04 April 2021

scrcp provides display and control of Android devices connected on USB (or over TCP/IP). It does not require any root access. It works on GNU/Linux, Windows and macOS.

[root@localhost ~]# cat /etc/fedora-release 
Fedora release 33 (Thirty Three)

[root@localhost ~]# vim /etc/yum.repos.d/scrcpy.repo 
[copr:copr.fedorainfracloud.org:zeno:scrcpy]
name=Copr repo for scrcpy owned by zeno
baseurl=https://download.copr.fedorainfracloud.org/results/zeno/scrcpy/fedora-$releasever-$basearch/
type=rpm-md
skip_if_unavailable=True
gpgcheck=1
gpgkey=https://download.copr.fedorainfracloud.org/results/zeno/scrcpy/pubkey.gpg
repo_gpgcheck=0
enabled=1
enabled_metadata=1
~
:x

install 

[root@localhost ~]# dnf install scrcpy -y

run 

[root@localhost ~]# scrcpy -b 24M --max-fps 60

list the available encoders

[root@localhost ~]# scrcpy --encoder _
INFO: scrcpy 1.17 <https://github.com/Genymobile/scrcpy>
adb: error: failed to get feature set: no devices/emulators found
ERROR: "adb push" returned with value 1

to record the screen while mirroring:

[root@localhost ~]# scrcpy -r file.mkv
