Advanced Package Tool, or APT

Tue 08 June 2021

The apt package is of "important" priority in all current Debian releases, and is therefore included in a default Debian installation. 

root@localhost:~# cat /etc/debian_version 
10.9
root@localhost:~# dpkg -s apt | egrep 'Desc|Ver'
Version: 1.8.2.3
Description: commandline package manager

apt-mark

performance of vlc is turning bad these days, let's make it won't upgrade while apt upgrade is excuting.

root@localhost:~# apt-mark hold vlc
vlc set on hold.
root@localhost:~# apt-mark showhold
vlc
root@localhost:~# apt-mark unhold vlc
Canceled hold on vlc.

apt list

-i means --installed, -u means --upgradeable, -a means --all-versions.

root@raspberrypi:~# apt list -iua
...

list all packages with -a, no matter installed or not.

root@raspberrypi:~# apt list -a vlc
Listing... Done
vlc/testing 3.0.12-0+deb10u1+rpt2 armhf [upgradable from: 3.0.12-0+deb10u1+rpt1]
vlc/now 3.0.12-0+deb10u1+rpt1 armhf [installed,upgradable to: 3.0.12-0+deb10u1+rpt2]
vlc/stable 3.0.12-0+deb10u1 armhf
