automatically load enviroment variables for debian user

Tue 08 December 2020

everytime u need to execute command source /etc/profile after logining to all new freshed debian server, u feel very inconvenient.

debian@hongkong:~$ cat /etc/debian_version 
10.7

let's look at /etc/profile for PATH variable

debian@hongkong:~$ head /etc/profile 
# /etc/profile: system-wide .profile file for the Bourne shell (sh(1))
# and Bourne compatible shells (bash(1), ksh(1), ash(1), ...).

if [ "`id -u`" -eq 0 ]; then
  PATH="/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/sbin:/bin"
else
  PATH="/usr/local/bin:/usr/bin:/bin:/usr/local/games:/usr/games"
fi
export PATH

ok, the final step to do is add PATH statement to /home/user/.bashrc

debian@hongkong:~$ vim .bashrc
...
if [ "`id -u`" -eq 1000 ]; then
  PATH="/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/sbin:/bin"
else
  PATH="/usr/local/bin:/usr/bin:/bin:/usr/local/games:/usr/games"
fi
export PATH
...
