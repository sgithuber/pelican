linux crontab job

Fri 07 May 2021

as usual, all new installed linux system will show result like below

# ┌───────────── minute (0 - 59)
# │ ┌───────────── hour (0 - 23)
# │ │ ┌───────────── day of the month (1 - 31)
# │ │ │ ┌───────────── month (1 - 12)
# │ │ │ │ ┌───────────── day of the week (0 - 6) (Sunday to Saturday;
# │ │ │ │ │                          7 is also Sunday on some systems)
# │ │ │ │ │
# │ │ │ │ │
# * * * * * <command to execute>

for better understand and remember, below are some examples:

  • it's someone's birthday, this script will only run once at 00:00 am 1th November.
0 0 1 11 * echo "happy birthday to him."
  • say "merry christmas" at 00:00 am of the day 24 and 25 in Decembar.
0 0 24,25 12 * echo "merry christmas"

  • / means every stuff.

  • - means from ~ to ~.
